# syslog-analyzer

A powerful shell script to analyze system logs (syslogs) on linux.

## Usage:
1. ```git clone https://github.com/iwtga/syslog-analyzer.git```
2. ```cd syslog-analyzer```
3. ```chmod +x sysloganalyzer.sh```
4. ```sudo ./sysloganalyzer.sh```
